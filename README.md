# get_http_file
Get file from http or https server with C++ qt implement

CHINESE

简要说明
这个项目主要实现文件下载功能。
1.支持http服务器的下载；
2.支持https服务器的下载；
3.支持断点续传；
4.可以判断自动跳转下载方式；

使用C++ Qt库完成
Qt版本是5.7.1
依赖库openssl
